Docker Installation

Docker provides a variety of commands to manage containers, images, and more. Some common commands are:

Docker install script

    root@localhost :  curl https://get.docker.com | bash

Portainer install script

    root@localhost : docker run -d -p 9443:9443 --name portainer --restart=always -v /var/run/docker.sock:/var/run/docker.sock -v portainer_data:/data portainer/portainer-ce:latest

Docker Compose

    root@localhost : sudo apt install docker-compose -y
